# 3D Game

OpenGL 3D rendering engine with Phong lighting.

## Build

```bash
mkdir cmake-build-debug
cd cmake-build-debug
cmake ..
cmake --build .
```

## Controls

- **WASD** - Move
- **Left Mouse** - Rotate camera
- **Middle Mouse** - Pan
- **Scroll** - Zoom
- **ESC** - Exit

## Tech Stack

C++20, OpenGL 4.3, GLFW, GLM, Assimp
